Aulas práticas — fontes de dados meteorológicos

As aulas práticas deste módulo replicam, em código Python, o conjunto de exercícios apresentados no Módulo 1 do curso Fenômenos Meteorológicos Adversos na Agricultura — Geotec. Cada página corresponde a um caderno Jupyter executado em aula, com o mesmo recorte de dados, as mesmas estações INMET, as mesmas COPs (Comunicações de Ocorrência de Perdas) do Sicor e os mesmos produtos de grade (MERGE/CPTEC, SAMeT/CPTEC) usados nos slides.

Nota

Os notebooks originais foram desenvolvidos para o Google Colab, acessando os dados a partir do Google Drive (/content/drive/MyDrive/MaterialGeotec/basededados/). Nesta documentação trocamos esse caminho por uma variável DATA_DIR no topo de cada exemplo — adapte para o seu ambiente local ou para o pacote disponibilizado no Kaggle.

Trilha sugerida

1. Catálogo de fontes

Mapa de portais públicos de dados meteorológicos: estações, monitores de seca, centros regionais e nacionais, grades de reanálise.

Catálogo de fontes de dados meteorológicos
2. Primeiros passos com Python e INMET

Mistura de Bash e Python no notebook; download massivo de CSVs horários do INMET (2014–2025); leitura com pandas.

Aula prática 1 — download e leitura dos dados do INMET
3. Sicor + INMET — plausibilidade espacial

Limpeza das coordenadas do Sicor, mapa de COPs recorrentes em Folium, busca das três estações INMET mais próximas via Haversine, balanço hídrico por fase fenológica da soja.

Aula prática 2 — Sicor + INMET: plausibilidade espacial e GDD da soja
4. Chuva excessiva em trigo

Variação do exercício anterior para trigo: percentual da chuva do evento sobre a safra, GDD do trigo (Tb=5°C).

Aula prática 3 — chuva excessiva em trigo: % do ciclo e GDD
5. Download dos dados de grade do CPTEC

Loops em Bash e Python para baixar climatologia e dados diários de MERGE (precipitação) e SAMeT (Tmin/Tmax) via FTP.

Aula prática 4a — download das grades do CPTEC (MERGE e SAMeT)
6. Manipulando dados em grade (xarray + Cartopy)

Leitura de NetCDF/GRIB2, cálculo de anomalias por dayofyear, séries temporais em pontos, mapas com Cartopy.

Aula prática 4b — manipulando dados em grade com xarray e Cartopy
7. Satélite GOES-19 (ABI, GLM, LST)

Conversão radiância → temperatura de brilho, reprojeção da grade geoestacionária para WGS84, raios (GLM) e LST.

Aula prática 4c — dados do satélite GOES-19 (ABI, GLM e LST)
