Plataformas de visualização de dados meteorológicos
====================================================

Levantamento de **plataformas de visualização** — viewers, mapas interativos e dashboards web —
de dados meteorológicos, hidrológicos e agrometeorológicos usáveis no Brasil, do nível estadual
ao global. Organizado por tipo de plataforma (G1–G6).

Este documento complementa :doc:`panorama-nacional` (fontes de dados brutos) e
:doc:`instituicoes-por-estado` (instituições). O foco aqui é **onde e como visualizar** os dados
de forma interativa — não os dados em si.

.. note::

   URLs e descrições confirmadas por acesso/busca (junho/2026). Campos não confirmados em
   fonte primária estão marcados ``[verificar]``. Plataformas *JS-heavy* (Simepar, SIGMA, INMET)
   foram verificadas por busca — funcionam no navegador mesmo que WebFetch não extraia
   conteúdo dinâmico.

**Ficha por plataforma:** operador/instituição · tipo · o que visualiza · cobertura ·
acesso (gratuito/login/pago) · URL.

G1 — viewers de satélite
---------------------------

Plataformas de visualização de imagens de satélite meteorológico (geoestacionário/polar) utilizáveis para o Brasil, incluindo nível global. Fichas intermediárias verificadas via WebSearch/WebFetch em 2026-06-28. Itens não confirmados marcados com **[verificar]**.

CIRA SLIDER
~~~~~~~~~~~

Visualizador interativo full-resolution ("Satellite Loop Interactive Data Explorer in Real-time") desenvolvido pelo CIRA/RAMMB. Permite ver todos os canais do ABI/AHI, comparar loops lado a lado e compartilhar o estado via URL.

.. list-table::
   :header-rows: 1
   :widths: 25 75

   * - Campo
     - Conteúdo
   * - Plataforma
     - CIRA SLIDER (e SLIDER-Archive para acervo)
   * - Operador/Instituição
     - CIRA + RAMMB (NOAA/NESDIS), Colorado State University (CSU)
   * - Tipo
     - Viewer de satélite (geoestacionário, tempo real e arquivo)
   * - O que visualiza
     - 16 bandas do ABI (GOES-East/West) e do AHI (Himawari); GeoColor e produtos value-added; setores Full Disk, CONUS, mesoescala; inclui produtos GLM (raios)
   * - Cobertura geográfica
     - Global via múltiplos geoestacionários; setor GOES-East cobre a América do Sul/Brasil
   * - Acesso
     - Gratuito, sem login
   * - URL
     - https://slider.cira.colostate.edu/ — arquivo: https://slider-archive.cira.colostate.edu/

NASA worldview / GIBS
~~~~~~~~~~~~~~~~~~~~~

Visualizador global da NASA Earthdata que consome o serviço GIBS (Global Imagery Browse Services). Permite navegar, comparar, animar e baixar mais de 1.200 produtos, muitos disponíveis em até ~3 h da observação.

.. list-table::
   :header-rows: 1
   :widths: 25 75

   * - Campo
     - Conteúdo
   * - Plataforma
     - NASA Worldview (frontend) / GIBS (backend de imagens)
   * - Operador/Instituição
     - NASA Earthdata (EOSDIS)
   * - Tipo
     - Viewer global de satélite (sobretudo polar: Aqua, Terra, VIIRS; também camadas geoestacionárias e produtos derivados)
   * - O que visualiza
     - +1.200 camadas: true color, combinações de bandas, detecções de incêndios ativos, aerossóis, temperatura, neve/gelo, overlays de costas/fronteiras/estradas
   * - Cobertura geográfica
     - Global (cobre o Brasil)
   * - Acesso
     - Gratuito, sem login; GIBS com API aberta (WMTS/WMS)
   * - URL
     - https://worldview.earthdata.nasa.gov/ — docs GIBS: https://nasa-gibs.github.io/gibs-api-docs/

G2 — mapas interativos e de modelos
-------------------------------------

Levantamento (G2) de plataformas de visualização do tipo **mapas interativos multi-camada** e **viewers de modelos de previsão** (incluindo modelos globais GFS/ECMWF/ICON). Dados confirmados por verificação web (WebSearch/WebFetch) em junho de 2026. Marcações ``[verificar]`` indicam pontos não confirmados diretamente nesta rodada.

Windy (windy.com)
~~~~~~~~~~~~~~~~~

Plataforma de mapa de vento e previsão interativa, uma das mais populares mundialmente. Animação de partículas de vento, dezenas de camadas e comparação de modelos.

.. list-table::
   :header-rows: 1
   :widths: 25 75

   * - Campo
     - Conteúdo
   * - Plataforma
     - Windy
   * - Operador/Instituição
     - Windyty, S.E. (empresa tcheca, fundada por Ivo Lukačovič, 2014; Praga). Detém participação majoritária na meteoblue AG
   * - Tipo
     - Mapa interativo multi-camada + viewer de modelos
   * - O que visualiza
     - Vento, rajadas, temperatura, chuva/trovoadas, nuvens, pressão, ondas/swell, CAPE, qualidade do ar, radar, satélite, etc. Modelos: globais **ECMWF, GFS, ICON**; locais NEMS, AROME, UKV, ICON-EU, ICON-D2, NAM, HRRR, ACCESS
   * - Cobertura geográfica
     - Global (modelos globais) + regionais
   * - Acesso
     - Freemium — todos modelos/camadas grátis; Premium (assinatura ~US$18,99/ano ou compra única ~US$29,99/1 ano) dá atualização 4x/dia e passos horários
   * - URL
     - https://www.windy.com

Agritempo — sistema de monitoramento agrometeorológico
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Sistema web mantido pela Embrapa (Embrapa Agricultura Digital / Informática Agropecuária) em parceria com o Cepagri/Unicamp. Operando desde 2003, disponibiliza gratuitamente boletins e mapas agrometeorológicos para todo o território nacional. Observações transversais
~~~~~~~~~~~~~~~~~~~~~~~~~~

- **Federais BR de referência:** INMET (Mapas/Satélite/Tempo), CPTEC/INPE (SIGMA, DSAT), REDEMET (radar),

CEMADEN (mapa interativo), ANA (Monitor de Secas) — todas gratuitas.

- **Globais abertas e robustas:** NASA Worldview, CIRA SLIDER, Windy, Ventusky, earth.nullschool, GDO.
- **Privadas/comerciais:** Wunderground/WunderMap, Climatempo, SOMAR, Meteologix, sigmameteorologia,

Climate FieldView — gratuitas em parte, com recursos sob assinatura.

- **Fontes subjacentes:** estas plataformas *consomem* os dados descritos em ``panorama-nacional``

(GOES, MERGE, ERA5, INMET, etc.) — visualização e dado bruto são camadas distintas.
