Caso 3 — geada em trigo

A geada é um risco crítico para o trigo, sobretudo nas fases de espigamento e floração [54, 55] (ver Agrometeorologia dos cultivos). Aqui a variável muda: em vez de chuva (MERGE), usa-se a temperatura mínima (TMIN) da grade SAMeT e das estações INMET. Uma COP por geada é plausível quando, na janela do evento, a TMIN ficou suficientemente baixa.

Raciocínio físico

A geada agrícola exige temperaturas próximas ou abaixo de 0 °C junto ao solo [84]. Isso permite um descarte físico direto: se a TMIN absoluta observada na janela do evento foi claramente alta (por exemplo, acima de 4 °C), a geada é fisicamente implausível — independentemente de qualquer anomalia. O caso combina, então, dois critérios: a anomalia de TMIN (esfriou menos que o normal?) e o valor absoluto de TMIN (esfriou o bastante para gear?).

Raciocínio do flag

#!/usr/bin/env python3
# Exemplo educacional — não operacional.
# Flag de plausibilidade para COPs de geada em trigo.
#
# Usa a temperatura mínima (TMIN, °C), não a chuva. Combina dois critérios:
#   (1) anomalia de TMIN não negativa (não esfriou mais que o normal); e
#   (2) descarte físico: TMIN absoluta na janela acima de 4 °C — alta demais
#       para gear. Os valores aqui são dados (sintéticos), para focar na regra.

# ============================================================
# IMPORTS
# ============================================================

import pandas as pd


# ============================================================
# CONFIGURAÇÕES
# ============================================================

# Acima deste valor de TMIN absoluta, geada é fisicamente implausível.
TMIN_LIMITE_GEADA = 4.0


# ============================================================
# DADOS DE EXEMPLO
# ============================================================

# tmin_evento_c : TMIN mínima absoluta observada na janela do evento (°C)
# anom_evento_c : anomalia de TMIN na janela (observado - climatologia, °C)
geada_trigo = pd.DataFrame({
    'ref_bacen':     ['C1',  'C2',  'C3',  'C4',  'C5'],
    'tmin_evento_c': [-2.5,   1.0,   8.0,   6.5,  -0.5],
    'anom_evento_c': [-4.0,  -1.5,   3.0,   2.0,  -3.5],
})


# ============================================================
# FLAGS DE PLAUSIBILIDADE
# ============================================================

# Flag 1: não esfriou mais que o normal (anomalia de TMIN >= 0).
geada_trigo['flag_geada_sem_anom_evento'] = (
    geada_trigo['anom_evento_c'].notna() &
    (geada_trigo['anom_evento_c'] >= 0)
)

# Flag 2: descarte físico — TMIN absoluta alta demais para gear.
geada_trigo['flag_geada_tmin_abs_alta'] = (
    geada_trigo['tmin_evento_c'].notna() &
    (geada_trigo['tmin_evento_c'] > TMIN_LIMITE_GEADA)
)

# Flag combinado: geada declarada, mas TMIN não baixou (anomalia) e/ou não
# chegou perto de zero (físico). O descarte físico basta para a suspeita.
geada_trigo['flag_geada_implausivel'] = (
    geada_trigo['flag_geada_tmin_abs_alta'] |
    (geada_trigo['flag_geada_sem_anom_evento'] &
     (geada_trigo['tmin_evento_c'] > 0))
)


# ============================================================
# DIAGNÓSTICO
# ============================================================

n_total = len(geada_trigo)
n_implausivel = int(geada_trigo['flag_geada_implausivel'].sum())

print(f'Total de COPs:              {n_total}')
print(f'Implausíveis (priorizar):   {n_implausivel}')
print('-' * 50)
print(geada_trigo[['ref_bacen', 'tmin_evento_c', 'anom_evento_c',
                   'flag_geada_implausivel']].to_string(index=False))

No material completo, esses critérios de grade são ainda corroborados pelas estações INMET mais próximas (a menor TMIN observada nas estações) e consolidados em um escore de plausibilidade de 0 a 100, que combina várias dimensões — anomalia, valor absoluto na grade e corroboração das estações. O escore é uma forma de ordenar os casos por grau de suspeita, do mais ao menos implausível.

Leitura

O descarte físico torna este caso particularmente robusto: uma geada declarada em uma janela com TMIN bem acima de zero é uma das incompatibilidades mais claras. Ainda assim, vale o mesmo princípio dos casos anteriores — o flag e o escore priorizam a análise; a confirmação de dano (que depende de fase fenológica, duração e intensidade da geada) é pericial e humana.
