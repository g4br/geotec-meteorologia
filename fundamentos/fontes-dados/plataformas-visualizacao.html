Plataformas de visualização de dados meteorológicos

Levantamento de plataformas de visualização — viewers, mapas interativos e dashboards web — de dados meteorológicos, hidrológicos e agrometeorológicos usáveis no Brasil, do nível estadual ao global. Organizado por tipo de plataforma (G1–G6).

Este documento complementa Panorama nacional de fontes de dados meteorológicos (fontes de dados brutos) e Instituições de meteorologia, hidrologia e agrometeorologia por estado (instituições). O foco aqui é onde e como visualizar os dados de forma interativa — não os dados em si.

Nota

URLs e descrições confirmadas por acesso/busca (junho/2026). Campos não confirmados em fonte primária estão marcados [verificar]. Plataformas JS-heavy (Simepar, SIGMA, INMET) foram verificadas por busca — funcionam no navegador mesmo que WebFetch não extraia conteúdo dinâmico.

Ficha por plataforma: operador/instituição · tipo · o que visualiza · cobertura · acesso (gratuito/login/pago) · URL.

Plataformas de visualização de imagens de satélite meteorológico (geoestacionário/polar) utilizáveis para o Brasil, incluindo nível global. Fichas intermediárias verificadas via WebSearch/WebFetch em 2026-06-28. Itens não confirmados marcados com [verificar].

CIRA SLIDER

Visualizador interativo full-resolution (“Satellite Loop Interactive Data Explorer in Real-time”) desenvolvido pelo CIRA/RAMMB. Permite ver todos os canais do ABI/AHI, comparar loops lado a lado e compartilhar o estado via URL.

Campo

Conteúdo

Plataforma

CIRA SLIDER (e SLIDER-Archive para acervo)

Operador/Instituição

CIRA + RAMMB (NOAA/NESDIS), Colorado State University (CSU)

Tipo

Viewer de satélite (geoestacionário, tempo real e arquivo)

O que visualiza

16 bandas do ABI (GOES-East/West) e do AHI (Himawari); GeoColor e produtos value-added; setores Full Disk, CONUS, mesoescala; inclui produtos GLM (raios)

Cobertura geográfica

Global via múltiplos geoestacionários; setor GOES-East cobre a América do Sul/Brasil

Acesso

Gratuito, sem login

URL

https://slider.cira.colostate.edu/ — arquivo: https://slider-archive.cira.colostate.edu/
